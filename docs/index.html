The Question

What if news headlines could predict
today's, this week's, and this month's chart mood?

The idea started from a hunch: music feels different during turbulent news cycles. When tensions rise — political crises, economic anxiety, global unrest — the charts seem to go harder, angrier, more intense. When things calm down, you get euphoric pop. Is that a real, measurable effect, or just pattern-matching noise?

To find out, I built a 14-module data pipeline from scratch. It ingests real-time data from six different sources, runs NLP emotion classification on thousands of news articles every week, clusters chart-topping songs by their sonic DNA, and trains a machine learning model to predict the following week's dominant audio mood from the news sentiment alone.

The result? 92% accuracy on the test set — and three AI-generated music clips: today's vibe, this week's sound, and this month's predicted mood.

The Data

Six Sources. One Pipeline.

Every source was chosen because it represents a different slice of the music-world relationship. No scraping of paywalled data — all public APIs, all reproducible.

📰
The Guardian API
News sentiment · Emotion classification

The backbone of the emotion analysis. Pulled 6,348 articles across topics like politics, technology, culture, and science — going back 16 weeks. Every article runs through a DistilRoBERTa transformer fine-tuned on 7 emotions.

REST API · HTTPS · Paginated · Daily batch pull
📊
Billboard Hot 100
Chart rankings · Weekly trends

The gold standard for US music popularity. Weekly snapshots of the Hot 100, including rank movement, weeks on chart, and peak position. Combined with Librosa audio features to understand the sonic profile of what's charting.

billboard.py library · Weekly pull · Historical backfill 3 months
🎵
iTunes + Last.fm
Global trending · Real-time listening

iTunes gives real-time chart rankings across 60+ countries. Last.fm adds global scrobble data — actual listening behaviour, not just purchases. Together they capture both commercial success and organic listening trends.

iTunes RSS · Last.fm REST API · Intraday polling
🎧
Librosa (Audio DNA)
30 acoustic features · Mood fingerprinting

Every trending track gets a 30-dimensional acoustic fingerprint: tempo, energy, 13 MFCCs, 12 chroma bins, spectral centroid, harmonic-to-percussive ratio. This is what the KMeans algorithm actually clusters — not song titles, pure sound.

Librosa Python · 30-second audio previews · Local processing
📺
YouTube Trending
Video virality · Cross-platform signal

YouTube's trending music videos act as a virality signal that's independent of streaming platforms. A song trending on YouTube before it hits Spotify is a leading indicator — useful for detecting mood shifts before they fully materialise on charts.

YouTube Data API v3 · Daily pulls · Multi-country
💬
Reddit (r/music + r/news)
Community sentiment · Engagement signals

Reddit captures what people are saying about music, not just what they're listening to. Score (upvotes minus downvotes) and comment counts serve as engagement proxies. A post going viral in r/music is often an early signal of a song breaking through.

PRAW (Python Reddit API Wrapper) · Streaming · OAuth2
The Architecture

14 Modules. End to End.

Built in layers — each module builds on the output of the previous. The pipeline runs daily, writes to BigQuery, transforms through dbt, and surfaces everything through a FastAPI backend.

Data Flow APIs / RSS→ Python Ingestion→ Google Cloud Storage→ BigQuery (raw)→ dbt (transformed)→ Python ML→ Pinecone + MusicGen→ FastAPI → Web
Ingestion Layer · Modules 1–8
M1 News + Reddit + YouTube — REST APIs, daily batch ingestion to GCS as JSONL, then loaded to BigQuery
M2 Spotify audio features — danceability, energy, valence, acousticness matched to chart tracks
M3–5 iTunes, Last.fm, Billboard — three chart sources merged into a unified trending_tracks table
M6 Librosa — 30-second previews processed locally, extracting 30 acoustic features per track
M7–8 dbt transformations — 15 staging views + 5 mart tables, 82 data quality tests
Analysis Layer · Modules 9–12
M9 DistilRoBERTa (j-hartmann model) — 7-emotion classification on 6,348 articles, batched in 64s, GPU-optional
M10 KMeans clustering — silhouette-scored, k=5 archetypes named by centroid profile: euphoric, melancholic, aggressive, peaceful, groovy
M11 Pearson correlation — 50 pairs (10 emotions × 5 moods), computed in both Python and BigQuery native CORR()
M12 XGBoost multiclass — 23 features (10 emotions + 5 mood pcts + 8 seasonal/temporal), class-balanced, 3-period inference: today · weekly · monthly
GenAI Layer · Module 13
13a Pinecone vector index — 1,011 tracks embedded as 30-dim acoustic vectors, StandardScaler fitted on the full corpus, cosine similarity search
13b MusicGen-small (Facebook/Meta) — runs 3× per pipeline (today/weekly/monthly), each with a season-aware blended prompt from mood probabilities + audio features, 500 tokens = 10s WAV
13c 3 WAVs (today/weekly/monthly) → GCS → BigQuery generated_tracks (period + mood_blend_json) → dbt staging → GitHub Pages history archive
Serving Layer · Module 14
API FastAPI backend — 8 endpoints querying BigQuery dbt_transformed views, CORS-enabled for cross-origin dashboard access
UI Two Streamlit dashboards — Mood Intelligence (ML analysis) and Music Trends (audio clustering) — plus this GitHub Pages static site
CI Prefect orchestration — daily 2am runs, all 14 modules in dependency order, failure alerts and retry logic

Does world sentiment actually correlate with music mood?

Each cell shows the Pearson r between a news emotion score (y-axis) and the percentage of trending songs assigned to a mood archetype (x-axis), across all 16 weeks. Red = positive correlation, blue = negative. Cells marked ★ are statistically notable (|r| ≥ 0.3). The strongest positive correlation tells you which emotion most reliably predicts which music mood — and the answer might surprise you. Look for the disgust → aggressive and neutral → aggressive pairs.

How do sentiment indices move week by week — and does the music follow?

The three lines are composite indices: Anxiety (fear + sadness), Tension (anger + disgust), and Positivity (joy + surprise). The dots on the right axis show what mood dominated the charts that week. This is the raw visual proof of the hypothesis. Look for weeks where anxiety or tension spikes — then check whether the music mood shifted the following week. That 1-week lag is exactly what the XGBoost model learns to predict.

Which emotions are the strongest predictors of each music mood?

Point-biserial correlation between each emotion feature and each mood archetype's binary label. Positive values mean "when this emotion is high, this mood tends to dominate." The standout finding: avg_neutral is the top driver of aggressive music — when news sentiment flattens out (neither joyful nor fearful, just numb), aggressive music fills the emotional void. Do different chart sources have different emotional personalities?

AI Generation
🤖

What does next week's music actually sound like?

The pipeline takes the XGBoost prediction, queries the Pinecone index for the 10 most sonically similar real tracks to that mood (by cosine similarity on 30 acoustic features), averages their audio DNA into a single profile, translates that profile into a natural language prompt ("aggressive rock music, high energy, 130 BPM…"), and feeds it to MusicGen-small to generate 10 seconds of original audio. The result is above — press play.
