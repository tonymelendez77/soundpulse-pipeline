Planned Migrations — Single Config Flag
Snowflake ──▶ BigQuery
WAREHOUSE=bigquery in .env
Triggered when 30-day trial ends.
Zero code changes required.
Databricks ──▶ Local MLflow
ML_TRACKING=local in .env
MLflow UI hosted on Render free tier.
Same experiment tracking interface.
Config Flag Pattern
# config/settings.py — the migration lives here from day one

WAREHOUSE = os.getenv("WAREHOUSE", "snowflake")   # flip to "bigquery"
ML_TRACKING = os.getenv("ML_TRACKING", "databricks") # flip to "local"

# config/warehouse.py
if WAREHOUSE == "snowflake":
    # connect to Snowflake (trial)
elif WAREHOUSE == "bigquery":
    # connect to BigQuery (production, free forever)
